# PassivesScan被动扫描之巅
漏洞扫描器一般分俩类：主动扫描和被动扫描。主动扫描一般是直接简单粗暴发起请求，被动扫描一般是根据代理的流量中特定参数进行简单粗暴的发起请求。有些漏洞通过主动扫描很难发现，必须通过被动扫描实现。被动扫描结合爬虫技术往往是漏洞扫描的最优解，这正是本项目准备解决的问题。作者：[0e0w](https://github.com/0e0w)

本项目创建于2021年12月28日，最近的一次更新时间为2022年5月20日。
